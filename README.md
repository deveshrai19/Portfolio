# 💼 React Portfolio

A modern and responsive personal portfolio website built with **React.js** to showcase my skills, projects, education, and contact information. The website is designed with a clean UI, smooth navigation, and a fully responsive layout.

## 🚀 Live Demo

🔗 https://deveshrai.netlify.app/

## 📂 GitHub Repository

🔗 https://github.com/deveshrai19/Portfolio

# 📖 About

This portfolio highlights my journey as a **Computer Science Engineering graduate** and showcases the projects, technologies, and skills I have learned. It provides recruiters and visitors with an easy way to explore my work and contact me.
